## Medical inventory management

_Medical inventory management is web based app use to manage and sell inventory of medical store which mainly consist of Authentication, Searching,Billing._

## Built With:

- Backend

1. `Python 3.6.7+`

#### Prerequisites

Your system must have Django3.0+.

- Frontend

1. Clone - `https://github.com/abdulrashid123/MedicalStoreManagement.git`
2. Install required packages using - `pip install`
3. Run Application : `python manage.py runserver`
